## Description

SunBnb is a intern project 

## Technologies used:

- Vue 3 (Compostion api)
- Laravel 10 
- Mysql
- Vuetify


## Prerequites:

- Node V20.4.0
- Npm V9.7.2
- Composer V2.5.8

## Manual Setup

### For FE
1. git clone `git@github.com:framgia/pj_sim-sunbnb.git`
2. cd pj_sim-sunbnb
3. cd fe
4. npm install
5. cp .env.example .env
6. npm run serve

### For BE
1. `cd be`
2. `composer install` 
3. cp .env.example .env
4. php artisan key:generate
5. php artisan serve
